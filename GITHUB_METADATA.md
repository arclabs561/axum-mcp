# GitHub Metadata Setup

Instructions for setting GitHub repository metadata.

## Repository Settings

### Description
Set in: GitHub → Settings → General → Repository details

**Value:**
```
HTTP-based MCP server framework for Rust
```

### Website
Set in: GitHub → Settings → General → Repository details

**Value:**
```
https://docs.rs/axum-mcp
```

### Topics
Set in: GitHub → Settings → Topics (or click "Add topics" on main page)

**Topics to add:**
- `rust`
- `mcp`
- `model-context-protocol`
- `axum`
- `http`
- `server`
- `ai`
- `agent`
- `framework`
- `async`

## Releases

### v0.2.0 Release

Create a GitHub Release (not just a tag):

1. Go to: https://github.com/arclabs561/axum-mcp/releases/new
2. **Tag**: `v0.2.0` (select existing tag)
3. **Title**: `v0.2.0`
4. **Description**: Copy from `CHANGELOG.md` section `[0.2.0]`
5. **Publish release**

See `CHANGELOG.md` for release notes content.
